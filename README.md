# คู่มือการรันระบบ Health Dashboard (Kiosk Mode)

โครงการนี้เป็นระบบจำลองข้อมูลสุขภาพ (อัตราการเต้นของหัวใจ, กราฟ EKG, น้ำหนัก) ออกแบบในสไตล์ธีมสีเทาแบบมินิมอล (Minimalist Gray Theme) มีความเปรียบต่างสูง (High Contrast) เพื่อการนำไปเปิดแสดงผลในโหมด Kiosk (เช่น บนเครื่อง Raspberry Pi)

---

## ขั้นตอนการสั่งเริ่มทำงาน (How to Run)

### 1. ติดตั้ง Library ต่าง ๆ (กรณีที่ย้ายโปรเจกต์หรือยังไม่ได้ติดตั้ง)
เปิด Terminal ในโฟลเดอร์นี้ แล้วรันคำสั่ง:
```bash
npm install
```

### 2. สั่งเริ่มระบบเซิร์ฟเวอร์จำลองการทำงาน (Start Development Server)
ใช้คำสั่งต่อไปนี้เพื่อเปิดเซิร์ฟเวอร์จำลองสำหรับดูผลลัพธ์แบบเรียลไทม์:
```bash
npm run dev
```
* เมื่อรันสำเร็จ ระบบจะให้ลิงก์สำหรับเปิดดู เช่น: **`http://localhost:5173/`**
* สามารถเปิดเว็บเบราว์เซอร์แล้วเข้าตามลิงก์ดังกล่าวเพื่อใช้งานได้ทันที

### 3. วิธีเปิดโหมดตู้บริการเต็มจอ (Fullscreen Kiosk Mode)
* **วิธีง่ายที่สุด (ผ่านหน้าเว็บ)**: คลิกปุ่ม **"Fullscreen Kiosk"** ที่อยู่บริเวณมุมขวาบนของหน้าจอ
* **วิธีเปิดอัตโนมัติบน Raspberry Pi หรือ Chromium**: รัน Chromium ผ่านคำสั่งเปิดเครื่องอัตโนมัติในโหมด Kiosk ด้วยคำสั่งนี้ใน Terminal:
  ```bash
  chromium-browser --noerrdialogs --disable-infobars --kiosk http://localhost:5173/
  ```

## สคริปต์ที่มีให้ใช้งาน (Available Scripts)

ในโครงการมีคำสั่งสำเร็จรูปให้ใช้งานผ่าน `npm run <command>` ดังนี้:

* `npm run dev` — เริ่มต้น Development Server บนเครื่องของคุณ (อัปเดตหน้าจออัตโนมัติเมื่อแก้โค้ด)
* `npm run build` — คอมไพล์โปรเจกต์เป็น HTML/JS/CSS พร้อมอัปโหลดขึ้นเว็บจริง (ไฟล์จะสร้างขึ้นในโฟลเดอร์ `dist/`)
* `npm run preview` — เปิดดูหน้าจอที่ผ่านการบิลด์เสร็จแล้วในเครื่องของคุณเพื่อทดสอบก่อนเอาขึ้นระบบจริง
* `npm run lint` — ตรวจสอบโครงสร้างและความถูกต้องของรูปแบบโค้ด
